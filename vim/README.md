## vim底线命令模式

```bash
:%# #\r#g # 空白换成换行符，在vim命令行模式下
:2,19s/aaa/bbb/g # 指定行号之间替换
:g/^$/d   # 把空白行删除，
:1        # 跳到文件首行
:$        # 跳到文件的尾行
:5p       # 跳转到第5行，然后显示第5行内容
:3d       # 跳转到第3行并删除第3行
: set nu  # 显示行号
: noh     # 除去高亮
```
